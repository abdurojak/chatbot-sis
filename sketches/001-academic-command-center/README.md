## Variant: Academic command center

### Design stance
Beranda diposisikan sebagai pusat kendali akademik: cepat discan, cepat di-tap, dan cepat dialihkan ke chat.

### Key choices
- Layout: hero besar di atas, dilanjutkan grid pintasan dan kartu status
- Typography: tegas dan ringkas agar terasa seperti dashboard mahasiswa
- Color: mengikuti biru utama aplikasi, dengan aksen emas untuk CTA chat
- Interaction: chip prompt, toast respons, dan toggle preview mode gelap

### Trade-offs
- Strong at: orientasi cepat, menunjukkan banyak kemampuan aplikasi, cocok untuk pengguna yang ingin langsung aksi
- Weak at: terasa lebih “tool-like” dibanding hangat; bisa terasa padat untuk first-time user

### Best for
- Mahasiswa yang sering bolak-balik cek KRS, pembayaran, MBKM, dan status wisuda
