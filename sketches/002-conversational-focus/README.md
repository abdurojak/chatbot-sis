## Variant: Conversational focus

### Design stance
Home diperlakukan sebagai perpanjangan dari chat: pengguna langsung melihat konteks percakapan, lalu memilih shortcut jika perlu.

### Key choices
- Layout: preview percakapan di atas, utilitas sekunder di bawah
- Typography: lebih ringan dan hangat, supaya terasa seperti assistant yang aktif
- Color: tetap pakai biru Trisakti, tetapi permukaan lebih lembut dan banyak ruang kosong
- Interaction: shortcut mengubah balasan preview, tab tools/timeline, dan composer bawah

### Trade-offs
- Strong at: onboarding, rasa personal, dan transisi mulus dari beranda ke chat
- Weak at: lebih sedikit informasi terlihat sekaligus dibanding varian dashboard

### Best for
- Pengguna yang datang untuk bertanya dulu, lalu baru masuk ke modul seperti KRS, pembayaran, atau MBKM
